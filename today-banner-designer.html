📘 사용 가이드

1. 단건 편집 모드

한 번에 1개 배너를 정성스럽게 편집·미리보기·다운로드할 때 사용합니다. 규격/언어를 바꾸면서 빠르게 시안을 비교할 수 있습니다.

2. 배치 생성 모드

리소스를 한 번만 입력하고, 생성할 규격(1:1 / 9:16 / 1200×628)과 언어(한/영/일/번체)를 체크하면 모든 조합을 자동 렌더링하여 ZIP으로 내보냅니다. v1.1부터 게임명·부제도 언어별로 각자 입력할 수 있어 로컬라이제이션이 필요한 상황에도 대응 가능합니다.

3. 확률형 고지 문구

한국어에만 *확률형 아이템 포함이 기본 노출됩니다. 영어/일본어/번체에서는 고지 필드가 비워져 있습니다 (필요 시 직접 입력 가능).

4. 리소스 준비 팁

  • 키아트: 배경 포함 1500px 이상 고해상도 PNG/JPG 권장. 카드 내부 배경으로만 사용됨
  • 오버플로우 (v1.2): 배경이 투명한 PNG로 SD 캐릭터만 크롭해서 업로드. 카드 경계를 넘어 튀어나오는 효과가 자동 적용됨
  • 앱 아이콘: 1:1 정사각형 512px 이상. 시스템이 자동으로 rounded square 처리

5. 오버플로우 레이어 (v1.2 신규)

키아트(카드 배경)와 오버플로우(카드 밖으로 튀어나오는 투명 PNG 캐릭터)는 별도로 업로드합니다. 업로드 후 사이드바의 슬라이더(X 오프셋 ±30%, Y 오프셋 ±15%, Scale 80%~120%)로 위치를 미세 조정할 수 있습니다. 규격을 변경하면 해당 사이즈 기본 프리셋이 자동 주입됩니다. 배치 모드에서는 사이즈별로 위치를 독립 저장하므로, 각 규격에 맞게 세밀히 튜닝 가능합니다. 레이어 순서는 키아트 카드 < 오버플로우 캐릭터 < 다크바/CTA/Today 워드마크로 고정되어 있어 UI 텍스트가 캐릭터에 가려지지 않습니다.

6. 매체 심사 주의

"Today" 워드마크는 Apple App Store 피처링 포맷을 차용한 것이므로, Google Ads/Meta 심사에서 일부 거절 사례가 있을 수 있습니다. 집행 전 매체별 승인 상태를 확인하세요.

7. 폰트

Pretendard(한/영), Noto Sans JP(일), Noto Sans TC(번체)가 CDN에서 로드됩니다. 모두 상업 사용 무료입니다. 오프라인 환경에서는 폰트가 제대로 렌더링되지 않을 수 있습니다.
