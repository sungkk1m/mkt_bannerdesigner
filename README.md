# Today Banner Designer — 사용 가이드

Apple App Store "Today" 스타일 배너를 표준화하여 빠르게 제작하기 위한 사내 전용 도구입니다. 레퍼런스 배너 2종(언더다크 : 디펜스, 네코맨서: 타워 디펜스)의 공통 패턴을 템플릿화하여 3가지 사이즈 × 4개 언어 조합으로 자동 출력합니다.

## 주요 기능

- 3가지 광고 규격 동시 지원 (1:1 / 9:16 / 1200×628)
- 4개 언어 팩 내장 (한국어 / English / 日本語 / 繁體中文)
- 한국어 버전에만 "*확률형 아이템 포함" 디스클레이머 자동 표기
- 단건 편집 모드 — 실시간 프리뷰로 한 개 타이틀 세밀 조정
- 배치 생성 모드 — 여러 게임 × 사이즈 × 언어 조합 일괄 출력
- **v1.2 신규**: **Overflow Layer** — 투명 PNG 캐릭터(SD 등)를 카드 영역 밖으로 튀어나오게 배치
- **v1.2 신규**: 키아트(카드 배경) · Overflow(캐릭터) **2-파일 분리 업로드**
- **v1.2 신규**: 사이즈별 **Overflow 프리셋 + X/Y/Scale 슬라이더**로 미세 조정
- **v1.1**: 배치 모드에서 게임명·부제를 **언어별로 개별 입력** (로컬라이제이션 지원)
- **v1.1**: 1200×628 레이아웃 리디자인 — 레퍼런스 기준 세로 스택 구조 + 우상단 프로필 아이콘
- PNG / JPEG 다운로드, 배치는 ZIP 일괄 저장
- 로컬 HTML 단일 파일 — 설치 없이 브라우저에서 바로 실행

## 파일 구성

- `today-banner-designer.html` — 본체 (단일 HTML 파일, 외부 CDN 4종 로드)
- `README.md` — 이 문서

외부 의존성은 모두 CDN으로 자동 로드되므로 파일만 열면 즉시 사용 가능합니다. 오프라인 환경에서는 CDN이 차단되어 동작하지 않을 수 있으니 주의하십시오.

## 시작하기

파일을 로컬로 내려받은 뒤 `today-banner-designer.html`을 더블클릭하면 기본 브라우저(Chrome / Edge 권장)에서 바로 열립니다. 별도 설치나 서버 구동이 필요하지 않습니다.

처음 진입하면 "단건 편집 / 배치 생성 / 도움말" 3개 탭이 상단에 보이며, 기본 탭은 단건 편집입니다.

## 단건 편집 모드

한 개 배너를 세밀하게 조정해서 바로 다운로드하는 모드입니다.

1. 왼쪽 입력 패널에서 프리셋 선택 (언더다크 / 네코맨서 / 직접 입력)
2. 타이틀 · 서브타이틀 · 언어 · 사이즈 선택
3. **키아트**(카드 배경) · **Overflow 캐릭터**(투명 PNG, 선택) · 앱 아이콘 이미지 업로드
4. Overflow 업로드 시 — 사이즈별 프리셋이 자동 적용되며, X(±30%) · Y(±15%) · Scale(80~120%) 슬라이더로 미세 조정
5. 오른쪽 프리뷰에서 실시간 확인
6. 하단 "PNG 다운로드" 또는 "JPEG 다운로드" 버튼 클릭

키아트는 `object-fit: cover` 방식으로 카드 영역에 꽉 채워집니다. Overflow 레이어는 **카드 경계 밖으로 튀어나올 수 있도록** 카드 위 z-index 10에 배치되며, 투명 PNG 캐릭터를 카드 상단 밖으로 배치할 때 사용합니다(레퍼런스의 SD 캐릭터 튀어나오는 연출).

## 배치 생성 모드 (v1.2 강화)

여러 조합을 한 번에 뽑아내고 싶을 때 사용합니다. 주간 캠페인 소재를 일괄 제작할 때 특히 유용하며, v1.1부터는 **언어별 독립 입력**, v1.2부터는 **사이즈별 Overflow 독립 튜닝**이 가능합니다.

1. 생성할 사이즈 체크박스 선택 (1:1 / 9:16 / 1200×628)
2. 생성할 언어 체크박스 선택 (ko / en / ja / zh-TW) — 체크된 언어만 아래 입력 필드에 표시
3. **프리셋 불러오기** (선택): 언더다크·네코맨서 등 프리셋 선택 → "모든 언어에 적용" 버튼 → 선택한 모든 언어에 게임명·부제 일괄 주입
4. **언어별 입력**: 각 언어에 대해 게임명, 부제, CTA, 고지문구를 독립적으로 입력 (현지화 필요 시 언어별로 다르게 입력)
   - 빠른 편의 기능: 한 언어에서 "다른 언어에 복사" 버튼을 누르면 해당 언어의 게임명·부제가 다른 체크된 언어에 자동 복사됨
5. 키아트 · Overflow 캐릭터 · 앱 아이콘 업로드 (전 언어 공통)
6. **사이즈별 Overflow 위치 튜닝** (v1.2): 사이즈 드롭다운에서 편집할 규격을 선택 → X/Y/Scale 슬라이더로 해당 사이즈 전용 값을 조정 → 선택한 각 사이즈마다 독립적으로 보정값을 저장
7. 파일명 Prefix 입력 (예: `underdark_today_v1`)
8. "ZIP 일괄 다운로드" 클릭 → 게임 × 사이즈 × 언어 경우의 수만큼 배너가 생성되어 ZIP 저장

예를 들어 사이즈 3개 × 언어 4개 = 12장이 한 번에 출력됩니다. 파일명은 `{prefix}_{size}_{lang}.png` 패턴으로 자동 명명됩니다.

**v1.1 이전(v1.0)과의 차이**: v1.0은 게임명·부제가 전 언어 공통이라 한국어 게임명이 영어/일본어에도 그대로 노출되었습니다. v1.1부터는 언어별로 번역된 게임명을 넣을 수 있어 글로벌 캠페인에 바로 쓸 수 있습니다.

## 지원 사양

사이즈는 1:1 (1080×1080, 정방형 피드·GDN), 9:16 (1080×1920, 세로 스토리·릴스), 1200×628 (가로 피드·디스플레이 광고) 세 가지입니다. 언어는 한국어 · 영어 · 일본어 · 번체중문 네 가지를 지원하며, 폰트는 Pretendard · Noto Sans JP · Noto Sans TC를 Google Fonts CDN에서 자동 로드합니다.

### 1200×628 레이아웃 (v1.1 리디자인)

v1.0에서 어색했던 가로 2분할 그리드 구조를 버리고, 레퍼런스 샘플 기준의 **세로 스택 레이아웃**으로 재구성했습니다.

- 상단: 큰 "Today" 워드마크 (이탤릭, 좌측 정렬)
- 우상단: 파란색 원형 프로필 아이콘 (App Store 스타일)
- 중앙: 키아트 카드 (라운드 코너, 28px)
- 하단: 검은 다크바 (앱 아이콘 · 게임명 · 부제 · CTA)
- 우측 최하단: 선택적 고지 문구

이 구조는 1:1·9:16 레이아웃과 기본 구조가 통일되어 있으며, CSS 변수로 사이즈별 폰트·패딩·간격만 미세 조정됩니다.

## Overflow Layer (v1.2 신규)

레퍼런스 배너(언더다크 : 디펜스, 네코맨서: 타워 디펜스)에서 SD 캐릭터가 **카드 상단 밖으로 튀어나오는 연출**을 재현하는 기능입니다.

### 구조

배너 내부는 3-layer z-index 구조로 분리되어 있습니다.

- **Card Layer (z:0)** — 키아트(배경 이미지). `overflow: hidden`으로 카드 영역 안쪽만 표시됩니다.
- **Overflow Layer (z:10)** — 투명 PNG 캐릭터. 카드 경계 밖으로 튀어나올 수 있습니다.
- **UI Layer (z:20)** — Today 헤더, 프로필 아이콘, 다크바, 고지 문구. Overflow 위에 항상 덮여 노출됩니다.

### 업로드 방식

키아트 슬롯과 Overflow 슬롯은 **별도의 파일 업로드**입니다.

- 키아트: 배경 포함 이미지(JPG/PNG 모두 가능). `object-fit: cover`로 카드에 꽉 채워짐.
- Overflow: 배경 없는 **투명 PNG 권장**. 카드 상단 위로 자연스럽게 바디 절반이 튀어나오는 형태.

Overflow가 업로드되지 않으면 해당 레이어는 렌더링되지 않으며, 키아트만 단독으로 노출됩니다.

### 사이즈별 프리셋

Overflow의 기본 위치·크기는 사이즈별로 다르게 세팅되어 있습니다(레퍼런스 기준).

| 사이즈 | X (좌우 이동) | Y (상하 이동) | Scale (크기) |
|---|---|---|---|
| 1:1 | 0% | -5% | 105% |
| 9:16 | 0% | -3% | 103% |
| 1200×628 | 5% | -2% | 102% |

- X: 양수 = 오른쪽, 음수 = 왼쪽
- Y: 음수 = 위쪽(카드 밖으로 튀어나옴), 양수 = 아래쪽
- Scale: 100% 기준 확대/축소

업로드 후 슬라이더로 미세 조정이 가능합니다. 단건 모드에선 단일 값, 배치 모드에선 **사이즈별 독립 값**이 저장됩니다.

## 법무 주의사항

본 도구는 레퍼런스 디자인의 구조적 특성만 추출하여 재현합니다. 실제 라이브 광고 집행 전 아래 사항을 반드시 확인하십시오.

- Apple "Today" 워드마크는 Apple Inc.의 상표입니다. 본 도구는 시각적 학습·내부 A/B 테스트용으로만 사용되며, Google Ads · Meta Ads · 기타 외부 매체 라이브 집행 시에는 매체 심사 정책 및 Apple 상표권 정책을 반드시 확인하십시오.
- 매체 심사 거절 리스크를 회피하려면 "Today" 대신 자사 브랜드 헤더로 교체한 버전을 병행 제작할 것을 권장합니다.
- 확률형 아이템 문구는 게임위 · 각국 규제 가이드라인에 따라 지역별 노출 정책이 다를 수 있으므로 법무팀 확인 후 집행하십시오.

## 알려진 제약

- 폰트 CDN(Google Fonts)이 차단된 환경에서는 시스템 폰트로 대체됩니다. 이 경우 레이아웃이 미세하게 달라질 수 있습니다.
- Internet Explorer는 지원하지 않습니다. Chrome · Edge · Safari · Firefox 최신 버전 사용을 권장합니다.
- 배치 생성 시 10개 이상의 대량 출력은 브라우저 메모리 제약으로 느려질 수 있습니다. 20개 이상은 배치를 나누어 진행하십시오.
